Termos de Uso

Ao usar o NavalhaOS, a barbearia concorda com estes termos.

1. Uso do sistema

O NavalhaOS é uma ferramenta para gestão de agenda, clientes, serviços, barbeiros, caixa, horários e comissões.

2. Acesso

O acesso é liberado apenas para contas ativas. O cadastro público não fica disponível no site.

3. Dados inseridos

A barbearia é responsável pelos dados cadastrados e pelas informações fornecidas aos clientes no momento do agendamento.

4. Disponibilidade

Podem ocorrer manutenções, instabilidades de internet, hospedagem ou serviços externos. O sistema deve ser usado como apoio à operação da barbearia.

5. Cancelamento

O plano pode ser cancelado conforme acordo comercial com o responsável pelo NavalhaOS. Contas bloqueadas deixam de acessar o painel.

6. Plano único

O NavalhaOS trabalha com plano único, com todas as funcionalidades disponíveis para clientes ativos.
